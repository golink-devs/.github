#  Welcome to Clink Devs

Clink Devs is an open-source collective dedicated to building high-performance, bare-metal backend infrastructure and networking tools. 

We build the robust systems that handle the heavy lifting. 

###  Featured Project

* **[CLink](https://github.com/clink-devs/clink)** - An ultra-fast, bare-metal Discord audio sending node (Lavalink alternative) written entirely in C for absolute maximum performance and zero garbage-collection overhead.
